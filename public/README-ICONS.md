# Icon Dosyaları

Bu klasöre aşağıdaki icon dosyalarını eklemeniz gerekmektedir:

## Gerekli Dosyalar

1. **icon.png** - Ana favicon (32x32 veya 16x16 piksel)
2. **apple-icon.png** - Apple touch icon (180x180 piksel)

## Icon Oluşturma

Icon dosyalarınızı şu şekilde oluşturabilirsiniz:

1. Online icon generator kullanın (örn: https://favicon.io, https://realfavicongenerator.net)
2. Kendi tasarımınızı kullanın
3. Basit bir logo veya metin kullanın

## Not

- Icon dosyaları `app/layout.tsx` dosyasında metadata'da tanımlanmıştır
- Next.js 14'te `app/icon.png` veya `app/icon.ico` dosyası da otomatik olarak favicon olarak kullanılır
- Eğer icon dosyalarını eklemezseniz, tarayıcı varsayılan favicon gösterecektir
